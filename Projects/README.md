# Projects — Data Analytics

A collection of complete data analytics projects organized by tool and platform, showcasing applied skills in data preparation, analysis, modeling, and visualization.

## Overview

This folder contains end-to-end analytical projects developed using industry-standard tools. Each project demonstrates the full workflow from raw data to actionable insights through structured analysis and interactive dashboards.

Projects are grouped by technology to highlight tool-specific capabilities and use cases.

## Folder Structure

### Excel/

Interactive Excel dashboards and analytical workbooks built from source data through cleaning, transformation, analysis, and visualization.

These projects demonstrate:

* Data preparation and transformation
* Advanced formulas and functions
* Pivot Tables and Pivot Charts
* Interactive filtering and slicers
* Dashboard design for business insights

---

### Power BI/

Comprehensive business intelligence reports developed using Power BI, including data modeling and advanced visualization.

These projects demonstrate:

* Data transformation using Power Query
* DAX calculations and measures
* Data modeling and relationships
* Interactive visuals and drill-down analysis
* End-to-end dashboard development

## How to Use

1. Navigate to the desired subfolder:

```bash
cd Projects/Excel
cd Projects/"Power BI"
```

2. Open files using the appropriate software:

* `.xlsx` files → Microsoft Excel
* `.pbix` files → Power BI Desktop

Each project is independent and can be explored individually.
