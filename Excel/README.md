# Excel Practice Files

A collection of topic-focused Excel workbooks created to develop and demonstrate proficiency in Microsoft Excel.

## Overview

This folder contains Excel files organized by specific concepts and features. Each workbook focuses on a particular topic, providing hands-on examples and exercises for building practical Excel skills.

Topics covered include:

* Data cleaning techniques
* Conditional formatting
* Chart creation and visualization
* Financial data examples
* Core Excel functions
* Advanced functions such as XLOOKUP and LAMBDA
* Common and specialized formula usage

## Tools Used

* **Microsoft Excel**
* Built-in formulas and functions
* Data formatting and visualization features

## How to Use

1. Clone the main repository:

```bash
git clone https://github.com/UmerAzmi/<your-data-analytics-repo>.git
cd <your-data-analytics-repo>/excel
```

2. Open any `.xlsx` file using Microsoft Excel.

Each file focuses on a specific concept and can be explored independently.
