# Data Analytics Repository

A comprehensive collection of data analytics work covering Excel and Power BI, including topic-focused practice files and complete real-world projects.

## Overview

This repository serves as a central hub for developing and demonstrating data analytics skills. It contains structured practice materials for building technical proficiency as well as end-to-end projects that apply those skills to realistic scenarios.

The work focuses on data preparation, analysis, modeling, visualization, and dashboard design to support insight generation and decision-making.

## Repository Structure

### Excel/

Topic-focused Excel workbooks designed to build strong fundamentals and advanced capabilities in spreadsheet-based analysis.

Key areas include:

* Data cleaning and preparation
* Formulas and advanced functions
* Conditional formatting
* Charts and visualization
* Financial data examples
* Specialized functions such as XLOOKUP and LAMBDA

---

### Power BI/

Concept-driven Power BI reports focused on core business intelligence capabilities.

Key areas include:

* DAX measures and calculated columns
* Power Query transformations
* Data modeling and relationships
* Drill-down analysis
* Interactive visual exploration

---

### Projects/

Complete end-to-end analytics projects demonstrating applied skills.

#### Projects/Excel/

Interactive Excel dashboards built from raw data through analysis and visualization.

Examples include sales analysis, HR insights, and business performance dashboards.

#### Projects/Power BI/

Full business intelligence reports developed using Power BI, including data modeling, transformation, and interactive dashboards.

Projects address realistic analytical scenarios such as retail analytics, transportation data, and operational reporting.

---

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/UmerAzmi/data-analytics.git
cd data-analytics
```

2. Navigate to any folder of interest:

```bash
cd Excel
cd "Power BI"
cd Projects/Excel
cd Projects/"Power BI"
```

3. Open files using the appropriate software:

* `.xlsx` files → Microsoft Excel
* `.pbix` files → Power BI Desktop
